# C3 Crunch Labs

This syllabus outlines a **10-week program** designed for Computer Science students at FIU, catering to all experience levels. The curriculum focuses on essential topics such as Agile methodologies, Java fundamentals, backend development of RESTful APIs using the Spring Boot framework, and frontend development with React Native. The program is structured to provide a comprehensive learning experience for students interested in full-stack development.

## 🔗 Quick Access  
- Repository Link: [C3 Crunch Labs](https://github.com/CODECRUNCHWORLDWIDE/C3-CRUNCH-LABS-PORTFOLIO)

## 📂 **Resources**  
This repository contains everything you’ll need to succeed in the program:  

- **Presentations**: Slide decks covering each week's topics.  
- **Scripts**: Sample code and scripts for backend and frontend development.  
- **Worksheets**: Hands-on activities to reinforce learning, including debugging, REST API design, and UI/UX exercises.
[C3 SPRING 2025](https://github.com/CODECRUNCHWORLDWIDE/C3-CRUNCH-LABS-PORTFOLIO/tree/main/SPRING-2025)

## 🛠️ Program Features  
The program complements the **Crunch Labs: Hackathon** sessions, providing students with practical, hands-on experience in full-stack development. Participants will have the opportunity to:
- Build **real-world projects** that integrate frontend and backend technologies.
- Apply concepts such as **Agile** development and **Java programming** to solve complex challenges.
- Use the **Spring Boot framework** for developing scalable RESTful APIs.
- Design intuitive user interfaces with **React Native** for mobile applications.
